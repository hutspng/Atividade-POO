# 1 O que é um vetor em C++? Explique com suas palavras.

Um vetor é uma estrutura de dados que acessa valores dentro de uma array (seja ele dinâmico ou estático) por meio de indices

# 2 Exemplo

temos uma array:
numero[{1,2,3,4,5}]
o vetor seria os indices desse array
o vertor pode percorrer os indices, para acessar os dados
```
for (int i=0;i<5;i++){
     std::cout << numero[i] << std::endl;
}
```
